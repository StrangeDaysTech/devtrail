# 贡献者契约行为准则

**语言**: [English](../../../CODE_OF_CONDUCT.md) | [Español](../es/CODE_OF_CONDUCT.md) | 简体中文

## 我们的承诺

作为成员、贡献者和领导者，我们承诺让参与 DevTrail 社区成为一种无骚扰的体验，无论年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种姓、肤色、宗教或性认同和性取向。

我们承诺以有助于建设开放、包容、多元、友好和健康社区的方式行事和互动。

## 我们的标准

### 有助于积极环境的行为示例：

- **尊重他人**：使用友好和包容的语言
- **富有同理心**：对其他社区成员表现出同理心
- **建设性态度**：给予并优雅地接受建设性反馈
- **勇于担当**：为错误承担责任，道歉并从中学习
- **协作精神**：关注对整个社区最有利的事情
- **保持耐心**：尤其是对新手和提问者

### 不可接受的行为示例：

- **骚扰**：使用性暗示的语言或图像，以及任何形式的性关注或性骚扰
- **恶意行为**：侮辱性或贬损性评论，以及人身攻击或政治攻击
- **歧视**：基于受保护特征的公开或私下骚扰
- **侵犯隐私**：未经明确许可发布他人的私人信息
- **不专业行为**：在专业环境中可被合理认为不当的其他行为
- **恐吓**：蓄意恐吓、跟踪或尾随（线上或线下）
- **干扰**：持续干扰社区活动，包括演讲和展示

## 执行职责

社区领导者负责阐明和执行可接受行为标准，并将针对他们认为不当、具有威胁性、冒犯性或有害的行为采取适当、公正的纠正措施。

社区领导者有权并有责任删除、编辑或拒绝不符合本行为准则的评论、提交、代码、Wiki 编辑、Issue 和其他贡献，并在适当时传达审核决定的理由。

## 适用范围

本行为准则适用于所有社区空间，包括：

- GitHub 仓库（Issue、Pull Request、讨论、Wiki）
- 官方沟通渠道
- 社区活动和聚会
- 代表项目时的社交媒体活动
- 与社区工作相关的一对一沟通

当个人在公共场合正式代表社区时，本行为准则同样适用。

## 举报

### 如何举报

如果你经历或目睹了不可接受的行为，或有任何其他顾虑，请通过以下方式联系项目团队进行举报：

**GitHub Issues**: [在此举报](https://github.com/StrangeDaysTech/devtrail/issues)

所有举报将被及时、公正地审查和调查。

### 应包含的内容

举报时，请包含：

- 你的联系方式
- 涉及的个人姓名（真实姓名、用户名或化名）
- 行为描述和背景
- 事件发生的日期和地点
- 任何可能有帮助的额外信息

### 保密性

所有社区领导者都有义务尊重举报者的隐私和安全。举报将以审慎和保密的方式处理。

## 执行指南

社区领导者将遵循以下社区影响指南来确定对任何违反本行为准则行为的后果：

### 1. 纠正

**社区影响**：使用不当语言或其他被认为不专业或在社区中不受欢迎的行为。

**后果**：社区领导者的私下书面警告，阐明违规性质以及为何该行为不当。可能会要求公开道歉。

### 2. 警告

**社区影响**：通过单次事件或一系列行为造成的违规。

**后果**：带有持续行为后果的警告。在指定时间内不与涉及的人员互动，包括不与执行行为准则的人员进行非自愿互动。这包括避免在社区空间以及社交媒体等外部渠道中的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内禁止与社区进行任何形式的互动或公开沟通。在此期间，不允许与涉及的人员进行公开或私下互动，包括不与执行行为准则的人员进行非自愿互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、对个人的骚扰，或对特定群体的攻击或贬低。

**后果**：永久禁止在社区内进行任何形式的公开互动。

## 申诉

如果你认为被不实或不公正地指控违反本行为准则，可以在 [GitHub Issues](https://github.com/StrangeDaysTech/devtrail/issues) 上开启详细的 Issue 进行申诉。申诉将由与做出原始决定不同的社区领导者审查。

## 归属

本行为准则改编自[贡献者契约](https://www.contributor-covenant.org) 2.1 版，可在 [https://www.contributor-covenant.org/version/2/1/code_of_conduct.html](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html) 查看。

社区影响指南受到 [Mozilla 行为准则执行阶梯](https://github.com/mozilla/diversity)的启发。

有关本行为准则的常见问题解答，请参见 [https://www.contributor-covenant.org/faq](https://www.contributor-covenant.org/faq)。
